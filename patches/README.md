## Pliki

Skompilowana paczka z wstrzykniętym tłumaczeniem PL jest publikowana w zakładce
**Releases** tego repozytorium.

Aktualny wariant testowy używa slotu tureckiego (`tr`) jako polskiego i wymaga dwóch
plików:

- `data.unity3d`
- `StreamingAssets/SaveFiles/LocalFiles/LocaDatabase.ltu`

Pobierz najnowszy release, rozpakuj zawartość payloadu do tego folderu (`patches/`),
a następnie uruchom `installers/linux/install-pl.sh`. Po instalacji wybierz w grze
język turecki.
